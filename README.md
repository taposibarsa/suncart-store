# 🌞 Suncart - Summer Essentials E-commerce Store

Suncart is a modern and responsive e-commerce web application focused on summer essential products. This platform allows users to browse, view, and purchase trending seasonal items with a smooth and user-friendly experience.

---

## 🚀 Live Demo
🔗 [(Live Link)](https://suncart-store-ebon.vercel.app/)

---

## ✨ Features

- 🛍️ Product listing with dynamic data
- 🔍 Single product details page
- 🔐 Authentication system (Login/Register)
- 🖼️ User profile update (name & image)
- 📱 Fully responsive design
- ⚡ Fast performance & optimized UI
- 🧭 Protected routes (only logged-in users can access certain pages)

---

## 🛠️ Tech Stack

**Frontend:**
- Next.js (App Router)

- Tailwind CSS
- daisyui

**Backend / Auth:**

**Tools & Libraries:**
